Die Programme in Dashboard STS1 sind für das Auslesen der Daten des Sateliten und für die Visualisierung der Daten in
einer Webapplikation zuständig.

!! Die neuste Version der Webaplikation hat ein Renderfehler der nach dem ersten Refresh der seite Auftritt. Dieser
Fehler betrifft nur den ersten Graphen. !!

Um die Webapplikation auszuführen muss zuerst unter DataLoad -> UDPSniff ausgeüfuhrt werden um empfangene Daten in eine 
CSV-Datei einzuschreiben. 

Anschließend kann dashboard.py ausgeführt werden.
